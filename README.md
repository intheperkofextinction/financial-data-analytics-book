
# Financial Data Analytics for Beginners

This repository contains the complete notebooks and datasets for the project
**“Financial Data Analytics for Beginners”**.

The goal of this project is to demonstrate practical financial data analysis skills,
including FP&A concepts, financial metrics, Python and SQL workflows,
and business-focused interpretation.

## Contents

- Business context and role of financial data analytics
- Financial statements and data foundations
- Financial metrics and ratio analysis
- Applying financial analysis using Python
- Applying financial analysis using SQL
- FP&A analysis (budgeting, forecasting, variance)
- Financial reporting and storytelling
- Real-world FP&A case study
- Analyst mindset and career guidance

## Dataset

A fictional but realistic FP&A-style dataset is included in the `data/` folder
and used throughout the case study chapter.

## Tools Used

- Python (pandas, numpy)
- SQL (analytical queries)
- Jupyter Notebook
- FP&A concepts and financial analysis techniques

## Purpose

This project is intended for learning, demonstration, and portfolio purposes.
